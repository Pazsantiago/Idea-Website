El blog sobre web's

Esta página va a abordar información de caracter 'objetivo' sobre los siguientes lenguajes:

Bienvenido

Me llamo Santiago, y el blog para dirigido para aquellos que esten comenzado en el camino del desarrollo web. En esta página se veran los temas mas básicos de una web como lo son los elementos necesarios para poder maquetarla e implementarla

Diseñador

Tenemos herramientas que nos permiten realizar un avance conceptual y solo de diseño preliminar de nuestra web ideal y final. Las mas usadas son Figma, Balsamiq ,Adobe Xd, entre otras.

Editor de texto

Programa en el cual uno puede hacer y ver aquello que se realiza en la codificación. Los mas conocidos son: Visual Studio Code, Sublime Text, Atom, Notepad, Bloc de notas, etc.

Navegador o buscador Web

Son Programas que nos permite ver la información o mas bien la página web en si misma. Con esto podremos ver tanto webs realizadas por nosotros como por otros. Los navegadores mas conocidos son: Google Chrome, Opera, Brave, Mozilla Firefox, Internet Explorer (aunque este ultimo sera retirado en 2022 y ya casi no es utilizado), etc.

Etiquetas

Ya a un nivel de escritura de codigo, son las partes que componen a la web. Son un conjunto de 'propiedades' o palabras que realizan cierta accion, y dentro de estas aplicaremos reglas que uno quiera, ya sea para mostrar algo o para asignar determinados funcionamientos, eventos, etc.

Hospedaje

El alojamiento web o web hosting es el servicio que provee a los usuarios de Internet un sistema para poder almacenar información, imágenes, vídeo, o cualquier contenido accesible vía web en internet o más específicamente en un servidor que por lo general hospeda varias aplicaciones o páginas web

Lenguajes

Son aquellas herramientas o mas bien formas de escribir codigo que nos permite especificamente crear y maquetar una web desde 0 separandolas en diferentes partes dependiendo de su 'funcionamiento'.

HTML: El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

Se le llama CSS u hoja de estilo en cascada (cascading style sheets) al lenguaje que usamos para definir desde el frontend, la presentación de un documento estructurado escrito en HTML. La idea principal que se encuentra detrás del desarrollo de CSS es separar la estructura de un documento de su presentación.

El frontend es toda la parte visible de las aplicaciones y sitios web. Esta área no trata directamente con bases de datos, servidores y todas las aplicaciones de back-end complejas, pero aborda la usabilidad, los efectos visuales y la velocidad de carga, entre otros detalles. Basicamente a través de las herramientas antes nombradas podemos formar una web desde el frontend para darle estructura y diseño a esta misma.
